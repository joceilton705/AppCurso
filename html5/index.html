HTML (HyperText Markup Language) É uma linguagem de marcação, capaz de inserir Novos conteúdos através de Hypertexto permitindo a construção de páginas web.
Neste curso utilizaremos o notepad ++ para desenvolver nossos projetos, assim o aluno poderá assimiliar melhor o conteúdo. baixar o Notepad ++

Para que nosso projeto funcione perfeitamente devemos utilizar uma estrutura html, vamos criar nosso primeiro documento html, não se preocupe caso não entender o que está fazendo nesse momento, o importante é termos um contato inicial com a linguagem html e vermos o primeiro resultado.

Primeiro Passo

Abra o Notepad ++ e tecle Control + N Para criarmos um novo arquivo

Segundo Passo

No menu no topo do programa clique em Linguagem vá até a letra H e selecione a opção HTML Para escolher a sintaxe da linguagem html
